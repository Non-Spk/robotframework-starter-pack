# robotframework-starter-pack

This project is a starter pack designed to be a foundational template for new projects. It includes a predefined directory structure, basic configuration, and essential setup scripts to accelerate the development process.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python >= 3.x
- pip >= 21.x
- Node >= 16.x

### Installation

1. **Create a virtual environment:**

    ```sh
    python -m venv .venv
    ```

2. **Activate the virtual environment:**

    **On Windows:**

    ```sh
    .\.venv\Scripts\activate
    ```

    **On macOS and Linux:**

    ```sh
    source .venv/bin/activate
    ```

3. **Install the required dependencies:**

    ```sh
    pip install -r requirements.txt
    ```

4. **Initialize Robot Framework Browser:**

    ```sh
    rfbrowser init
    ```
